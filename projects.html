Chorely (in progress) is a chore-tracking web application enabling households to manage their chores more efficiently. Work collaboratively with others in your household to assign and complete chores. Add upcoming events to the calendar. Read and write recipes, and more!

I started this app to make shared housework easier to organise and manage, instead of relying on having it all written out, the paper getting lost or damaged, people ignoring it, etc. If everyone in the household uses this app, everyone can see which chores are done, they can be automatically assigned, and if someone is away they can reassign chores to others.

    Features:
  • Login and logout
  • Chore and event tracking
  • Collaboration with other users
  • Assign and reassign chores
  • Add, delete, and update chores and events
  • Calendar
  • Recipes
  • Responsive and stylish UI
    Tech Stack:
  • Frontend - React + Vite
  • Backend - Node + Express
  • Database - SQLite (dev) / PostgreSQL (prod)
  • Authentication - Auth0
  • Images - Multer
  • Styling - Tailwind CSS + custom UI

A group project built in my final week at Dev Academy Aotearoa. I was responsible for building the lost pet submission forms, the databases, migrations, seeds, and implementing the image upload feature.

TailTrail is a full-stack web application that helps reunite lost pets with their owners. Users can report lost or found pets, view nearby pet sightings on an interactive map, and browse recent cases. Built with modern technologies like React, TypeScript, Node.js, and Google Maps API.

    Features:
  • Interactive Google Map with custom styling
  • Submit lost or found pet reports via form
  • Filter pets by status (Lost / Found / All)
  • View recent pet sightings in real-time
  • Auth0 authentication (Sign in / Sign out)
  • Beautiful responsive UI styled with Tailwind CSS
    Tech Stack:
  • Frontend - React + Vite
  • Backend - Node + Express
  • Database - SQLite (dev) / PostgreSQL (prod)
  • Authentication - Auth0
  • Maps - Google Maps API
  • Styling - Tailwind CSS + custom UI
